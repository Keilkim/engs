## 1. SRS 복습 엔진 (학습 핵심)
- **자체 'stack' 방식을 표준 SM-2로 교체** — 간격이 지수적으로 확장(1→6→15→38일…), `-10 영구제외` 폐기. 기존 스키마의 `ease_factor/repetitions/interval_days` 사용. 2버튼 UI 유지(앎=q4, 모름=q2).
- **타임존 수정** — 신규 `src/utils/dateUtils.ts`의 `localDateString()`로 모든 UTC 날짜 계산 교체(KST 자정~09시 카드 누락/1일 간격 무효화 해결).
- **죽은 통계 배선 복구** — 정답 시 `repetitions` 실제 증가 + `handleEvaluation`에서 `updateTodayStats` 호출 → 대시보드 'Reviewed'/'Rate'/주간 차트가 실제 반영. 집계 기준을 `last_reviewed IS NOT NULL`로 변경.
- **홈 카운트 ↔ 복습 큐 필터 완전 일치** — "N개 복습" → 빈 완료 화면 모순 제거.
- **인출 강제** — 답 공개(`showAnswer`) 후에만 평가 버튼 노출.
- **크래시 방지** — 평가 연타 재진입 차단, `currentIndex` 범위 방어, Flashcard의 `JSON.parse`를 `safeJsonParse` 폴백.
- **질문=답 카드 근절 + 중복 방지** — 분석 데이터(`isVocabulary`/`grammar`)가 있는 highlight만 review_item 생성, 동일 annotation 중복 생성 차단.
- **'Review Again' 허위 버튼 제거**, 신규 사용자용 빈 상태 CTA 추가.
